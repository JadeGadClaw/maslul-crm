# מסלול CRM — דמו ראשוני

אפליקציית ניהול מקומית בעברית ובכיוון ימין־לשמאל לחברת הסעות, שליחויות והובלה קלה שמחליפה עבודה בגיליון ידני.

## מה כלול

- כניסת מנהל לדמו עם קוד: `admin123`
- ממשק עברי מלא עם `lang="he"`, `dir="rtl"`, תצוגת שקלים וכותרות ייצוא בעברית
- ניהול נסיעות: תאריך, שעה, סוג רכב, סוג עבודה, נהג, נלקח ע״י, לקוח, חברה, נקודת איסוף, יעד, אזור, מחיר, סטטוס תשלום, סטטוס קבלה והערות
- השלמת מקומות חכמה מרשימת מקומות קבועה ומנסיעות קודמות
- יומן נסיעות עם סינון לפי תאריך, חברה, נהג, רכב, אזור, חודש, שנה וחיפוש חופשי
- ייצוא נתונים לקובץ גיליון עם כותרות עבריות
- לוח סיכום: הכנסות היום, החודש והשנה, חלוקה לפי רכבים, לקוחות מובילים ומסלולים נפוצים
- תבנית קבלה עברית להדפסה עבור נסיעה שנבחרה
- נתוני דמו בעברית וכפתור איפוס

## הפעלה מקומית

```bash
cd /srv/gad-share/apps/maslul-crm
npm install
npm run dev
```

פתיחה בדפדפן: `http://localhost:3021`.

## בדיקות

```bash
npm run typecheck
npm run build
```

הפקודה `next lint` עשויה לפתוח שאלון הגדרה אינטראקטיבי בפרויקט חדש של Next 15 אם עדיין אין הגדרת ESLint. הדמו לא יוצר הגדרות לינט אוטומטית.

## אחסון נתונים

בשלב הדמו, הנסיעות נשמרות בדפדפן תחת `maslul-crm.jobs.v1`. זה מתאים להצגה ולבדיקה, אבל עדיין לא מתאים לעבודה מרובת משתמשים.

שלבי הקשחה מומלצים:

1. התחברות אמיתית וניהול הרשאות
2. מסד נתונים בצד שרת
3. ייבוא מהגיליון הקיים
4. תפקידים למנהל, מוקדן, נהג והנהלת חשבונות
5. הפקת קבלות מתקדמת ושליחה ללקוח
